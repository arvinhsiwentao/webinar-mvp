# 核心賣點與獨特價值主張 (USP Analysis)

---

## Mike 品牌的核心 USP

### 一句話價值主張

> **「普通人也能複製的美股財務自由路徑」**

### 支撐這個主張的三大支柱

1. **真實軌跡**：Mike 本人從普通人到 43 歲財務自由的完整故事
2. **系統方法**：可複製的抄底 + 存股雙策略
3. **持續陪伴**：不只賣課程，還有 APP 工具和社群支持

---

## 各產品線的 USP

### 《破局致富》書籍

| USP | 說明 |
|-----|------|
| **真實人生故事** | 首次揭露 1980-2025 完整人生軌跡，不是理論家是實踐者 |
| **普通人視角** | 不是金融專業出身，用白話解釋複雜概念 |
| **認知升級** | 不只教投資技巧，更教思維模式改變 |
| **全球視野** | 美籍華人雙重視角，看懂中美兩地財富邏輯 |

**關鍵賣點句**：
- 「你賺不到認知以外的錢」
- 「一本普通人可以直接看懂的財經書」
- 「寫給想賺錢的你，也寫給 20 年前的我」

## 競爭優勢矩陣

### Mike vs 其他美股 KOL

| 維度 | Mike是麥克 | 一般 KOL |
|------|-----------|----------|
| **財務自由實績** | 43 歲達成，有具體時間軸 | 多為理論或未公開 |
| **產品生態系** | 書 + APP + 課程 + 社群 | 通常只有單一產品 |
| **機構合作** | CMoney 背書 | 個人為主 |
| **持倉公開** | 透明公開 | 少見 |
| **語言風格** | 平實、幽默、接地氣 | 多為專業術語 |
| **受眾關係** | 「朋友分享」 | 「老師教學」 |

### Mike 的獨特定位

```
                高專業度
                    │
                    │
        學術派 ─────┼───── Mike是麥克
                    │       (專業 + 親切)
                    │
                    │
        低專業度 ───┼───── 普通網紅
                    │
               低親切度        高親切度
```

## 情感賣點

### 理性賣點 (What)
- 4 年財務自由
- ETF 連續 10 年 15-20% 報酬
- 20 萬訂閱驗證
- 系統化策略工具

### 情感賣點 (Why)
- 「我也是普通人，你也可以」
- 「不再為錢焦慮」
- 「擁有選擇的自由」
- 「有人陪你一起走這條路」

### 行動賣點 (How)
- 清楚的步驟可以照著做
- 工具幫你省時間
- 社群有問題隨時問
- 免費試用沒有風險
