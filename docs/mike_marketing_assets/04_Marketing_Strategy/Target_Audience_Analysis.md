# 目標受眾分析 (Target Audience Analysis)

---

## 主要受眾區隔

### 區隔 A：美股新手探索者
**人口特徵**
- 年齡：25-35 歲
- 職業：上班族、小資族
- 收入：月薪 3-8 萬台幣
- 投資經驗：零到少許

**心理特徵**
- 對美股好奇，但不知從何下手
- 擔心風險，怕賠錢
- 時間有限，希望有人帶領
- 渴望被動收入，但不確定是否可行

**痛點**
- 「美股聽起來很複雜，我能懂嗎？」
- 「沒有很多錢，可以投資嗎？」
- 「怕選錯股票賠光本金」
- 「資訊太多，不知道該相信誰」

**對 Mike 產品的需求**
- 入門課程 / APP 的基礎功能
- 清楚的步驟指引
- 社群支持與問答

---

### 區隔 B：已有投資經驗的進階者
**人口特徵**
- 年齡：30-45 歲
- 職業：中階主管、專業人士、小企業主
- 收入：月薪 8 萬+ 或有一定積蓄
- 投資經驗：1-5 年

**心理特徵**
- 已經在投資，但績效不如預期
- 想學習更系統化的方法
- 對 ETF 有興趣，但不知道怎麼配置
- 願意付費學習，追求效率

**痛點**
- 「自己研究太花時間，想要有效率的方法」
- 「賺賺賠賠，沒有穩定獲利」
- 「想知道專家怎麼操作」
- 「想增加被動收入，但不知道下一步」

**對 Mike 產品的需求**
- 進階 ETF 課程
- APP 的抄底/存股策略工具
- Mike 的持倉清單與操作邏輯

---

### 區隔 C：追求財務自由的積極者
**人口特徵**
- 年齡：35-50 歲
- 職業：中高階主管、創業者
- 收入：高收入或有資產累積
- 投資經驗：5 年以上

**心理特徵**
- 明確想要達成財務自由
- 願意投入時間和金錢學習
- 追求系統化、可複製的方法
- 重視實戰經驗與社群交流

**痛點**
- 「工作很忙，需要更高效的投資方式」
- 「想要專家的即時觀點」
- 「希望有社群可以交流討論」
- 「想加速財務自由的進度」

**對 Mike 產品的需求**
- 完整的 APP 年訂閱
- 期權課程（進階收入策略）
- VIP 社群與語音直播

## 受眾共通的心理訴求

### 渴望
- 財務自由：不用為錢工作
- 被動收入：睡覺也能賺錢
- 安全感：不用擔心經濟波動
- 認同感：想被認為是「懂投資」的人

### 恐懼
- 賠光本金
- 錯過機會
- 被騙或被割韭菜
- 永遠無法擺脫「為錢工作」的循環

### 阻礙
- 資訊過載：不知道該學什麼
- 時間不足：沒時間研究
- 信任缺失：不知道誰值得信任
- 自我懷疑：覺得自己不夠聰明

### 主要管道
| 管道 | 受眾類型 | 內容形式 |
|------|----------|----------|
| YouTube | 所有區隔 | 教學影片、市場分析、Podcast |
| Facebook 社團 | B、C 區隔 | 社群討論、直播 |
| CMoney 平台 | B、C 區隔 | 課程、APP、文章 |
| Instagram / Threads | A 區隔 | 短影片、圖文懶人包 |

### 付費意願
- **A 區隔**：低價入門（書籍、免費試用）
- **B 區隔**：中價課程（NT$3,000-10,000）
- **C 區隔**：高價方案（APP 年訂閱 NT$17,000+）

## 受眾旅程地圖

```
認知 → 興趣 → 考慮 → 購買 → 忠誠
 │      │      │      │      │
 │      │      │      │      └─ VIP 社群、進階課程
 │      │      │      └─ 課程/APP 訂閱
 │      │      └─ Landing Page、Webinar
 │      └─ YouTube 影片、書籍
 └─ 社群口碑、搜尋
```

### 各階段觸點
1. **認知**：YouTube 影片、朋友推薦、搜尋「美股投資」
2. **興趣**：看更多影片、追蹤頻道、購買書籍
3. **考慮**：Landing Page、Webinar、免費試用
4. **購買**：課程、APP 訂閱
5. **忠誠**：VIP 社群參與、推薦朋友、購買更多產品

## 轉換關鍵洞察

### 什麼讓受眾決定購買？
1. **社會證明**：20 萬訂閱、2,000+ 會員、學員見證
2. **權威背書**：CMoney 合作、出版書籍
3. **真實故事**：Mike 從普通人到財務自由的歷程
4. **風險降低**：免費試用、退款保證、分期付款
5. **稀缺性**：早鳥優惠、限時折扣

### 什麼阻礙受眾購買？
1. 價格疑慮：「太貴了」
2. 信任疑慮：「會不會是割韭菜？」
3. 能力疑慮：「我能學會嗎？」
4. 時間疑慮：「我有時間嗎？」
5. 效果疑慮：「真的有用嗎？」
