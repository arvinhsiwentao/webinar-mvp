# 銷售異議處理指南 (Objection Handling)

---

## 常見異議與回應策略

### 1. 價格異議：「太貴了」

**異議類型**
- 「課程/APP 太貴，我負擔不起」
- 「市面上有免費的資訊，為什麼要付費？」
- 「不確定值不值這個價格」

**回應策略**

| 異議 | 回應 |
|------|------|
| 太貴 | 「這個價格換算下來，每天不到 XX 元。如果能幫你少走 10 年彎路，這筆投資的回報是多少？」 |
| 有免費的 | 「免費的資訊需要你自己篩選、整理、驗證。Mike 已經幫你做好這些，你付的是時間和試錯成本的節省。」 |
| 不確定值不值 | 「所以我們提供免費試用，你可以先體驗看看是否適合你。」 |

**支撐論點**
- 分期付款選項：「影音課程滿 3 千適用分期 0 利率」
- 價值比較：「一次錯誤的投資決策可能損失幾萬甚至幾十萬」
- ROI 框架：「如果學到的方法讓你多賺 5%，這筆學費很快就回本」

---

### 2. 信任異議：「會不會是割韭菜？」

**異議類型**
- 「網路上太多詐騙，怎麼知道這是真的？」
- 「他真的有賺到錢嗎？」
- 「為什麼要教別人，自己賺就好了？」

**回應策略**

| 異議 | 回應 |
|------|------|
| 怕詐騙 | 「Mike 與 CMoney（上市公司）合作，有機構背書。他的 YouTube 頻道有 20 萬訂閱，2000 萬次觀看，如果是詐騙早就被揭穿了。」 |
| 真的賺錢嗎 | 「他在 APP 裡公開自己的持倉清單，連買了什麼都透明。書裡也詳細寫了他從負債到財務自由的完整軌跡。」 |
| 為什麼要教 | 「他已經財務自由了，分享知識是他現在的事業，也是幫助更多普通人的方式。這跟他投資賺錢不衝突。」 |

**支撐論點**
- CMoney 合作背書
- 暢銷書出版（經過出版社審核）
- 公開持倉清單（透明度）
- 長期經營的 YouTube 頻道（不是突然冒出來的）

---

### 3. 能力異議：「我能學會嗎？」

**異議類型**
- 「我不懂英文，能投資美股嗎？」
- 「我沒有金融背景」
- 「我很忙，沒時間學」

**回應策略**

| 異議 | 回應 |
|------|------|
| 不懂英文 | 「Mike 的 APP 和課程都是全中文。美股投資不需要你看英文，工具會幫你處理。」 |
| 沒有背景 | 「Mike 自己也不是金融專業出身。他說：『這本書寫給普通人可以直接看懂。』」 |
| 沒時間 | 「這就是為什麼要用工具。APP 的燈號系統和策略篩選，幫你省下大量研究時間。」 |

**支撐論點**
- Mike 自己的背景（普通人出身）
- 課程設計（從基礎開始）
- 工具設計（省時間、降門檻）
- 社群支持（有問題可以問）

---

### 4. 效果異議：「真的有用嗎？」

**異議類型**
- 「學了就能賺錢嗎？」
- 「別人成功不代表我會成功」
- 「投資不是都有風險嗎？」

**回應策略**

| 異議 | 回應 |
|------|------|
| 學了就能賺 | 「投資沒有人能保證獲利。但正確的方法可以提高成功機率，減少犯錯。Mike 的方法是他 10 年驗證過的。」 |
| 別人成功 | 「所以 Mike 強調的是『可複製的方法』，不是靠運氣。他的策略是系統化的，你可以照著做。」 |
| 風險 | 「對，所以 Mike 的理念是『守住本金』優先。他不教一夜暴富，教的是穩定累積。」 |

**支撐論點**
- 不承諾暴富（誠實）
- 強調方法論（可複製）
- 風險管理優先（守住本金）
- 長期視角（複利效應）

---

### 5. 時機異議：「現在不是好時機」

**異議類型**
- 「市場不好，現在不適合投資」
- 「等我存夠錢再說」
- 「等有空再學」

**回應策略**

| 異議 | 回應 |
|------|------|
| 市場不好 | 「市場永遠有人說不好。但 Mike 的抄底策略就是專門在市場低迷時找機會。最好的買點往往是別人恐懼的時候。」 |
| 存夠錢 | 「美股可以一股一股買，不需要很多錢。重點是先開始學習，準備好機會來時能抓住。」 |
| 有空再學 | 「複利的關鍵是時間。早一年開始和晚一年開始，十年後的差距是巨大的。」 |

**支撐論點**
- 震盪行情策略（任何市場都能操作）
- 低門檻入手（一股一股買）
- 複利效應（時間越早越好）
- 學習本身是投資（先學後用）

## 異議預防：Landing Page 必備元素

### 在異議出現前就解答

1. **價格異議預防**
   - 顯示原價與折扣價對比
   - 分期付款選項
   - 價值比較（vs 試錯成本）
   - 免費試用 CTA

2. **信任異議預防**
   - Mike 的完整背景介紹
   - CMoney 合作標誌
   - 學員見證 / 評價
   - 媒體報導（如有）

3. **能力異議預防**
   - 「適合零基礎」的標示
   - 課程大綱預覽
   - 「普通人也能懂」的文案
   - 社群支持說明

4. **效果異議預防**
   - 具體成果數據（20萬訂閱、10年績效）
   - 方法論說明（不是靠運氣）
   - 學員案例（如有）
   - 合理的期望設定（不承諾暴富）

## Webinar 異議處理時機

| 時間點 | 處理的異議 |
|--------|-----------|
| 開場 | 建立信任（自我介紹、故事） |
| 中段 | 能力異議（強調方法可學習） |
| 效果異議（展示案例） |
| 結尾 | 價格異議（揭示優惠）|
| 時機異議（限時稀缺性）|
| Q&A | 所有殘餘異議 |
