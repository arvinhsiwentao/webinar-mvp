# Landing Page & Webinar 內容框架

---

## Landing Page 結構建議

### Hero Section（首屏）

**必備元素**
- 主標題：痛點 + 解決方案
- 副標題：具體承諾
- CTA 按鈕：明確行動
- Mike 照片或影片
- 社會證明數據（1-3 個）

**文案範例**
```
主標題：「還在為錢工作？Mike 教你 4 年達成財務自由」
副標題：「20 萬人追蹤的美股投資專家，公開他的抄底存股雙策略」
CTA：「免費報名 Webinar」或「立即試用 APP」
數據：「43 歲財務自由 ｜ 20 萬訂閱 ｜ 2000+ 會員」
```

---

### Problem Section（痛點）

**目標**：讓受眾感覺「這就是在說我」

**常見痛點清單**
1. 工作很累但存不到錢
2. 想投資但怕賠錢
3. 資訊太多不知道從何開始
4. 看別人賺錢但自己不會
5. 擔心老了沒有被動收入

**文案框架**
```
你是不是也有這些困擾？
☐ 每天辛苦工作，但存款增加的速度追不上物價
☐ 想投資美股，但看到滿滿的英文就頭痛
☐ 看過很多投資書，但還是不知道怎麼下手
☐ 擔心選錯股票，賠掉辛苦存的錢
☐ 羨慕別人的被動收入，但不知道從何開始
```

---

### Solution Section（解決方案）

**目標**：介紹 Mike 的方法如何解決痛點

**結構**
1. 方法論簡介（What）
2. 為什麼有效（Why）
3. 怎麼運作（How）

**內容要點**
- 抄底策略：在市場恐懼時找到低價買點
- 存股策略：選擇穩定配息股建立被動收入
- 雙策略組合：長線資產配置 × 短線機會捕捉
- 工具支持：APP 燈號系統一眼判斷

---

### Credibility Section（可信度）

**Mike 的故事簡版**
```
從負債累累到 43 歲財務自由

2018：開始投資美股，成為特斯拉早期投資者
2022：4 年內達成財務自由
2023：創立 YouTube 頻道，累積 20 萬訂閱
2024：與 CMoney 合作推出【Mike是麥克】APP
2025：出版暢銷書《破局致富》

「我不是金融專業出身，我只是一個想賺錢的普通人。」
```

---

### Product Section（產品介紹）

**針對不同產品調整**

**APP Landing Page**
- 6 大功能逐一介紹（配圖/動圖）
- 價格比較表
- 免費試用 CTA

**課程 Landing Page**
- 課程大綱預覽
- 講師介紹
- 學員見證
- 早鳥優惠

**Webinar Landing Page**
- Webinar 主題和內容預覽
- 時間和參加方式
- 免費報名 CTA
- 講師介紹

---

### Social Proof Section（社會證明）

**必備**
- 3-5 則學員見證（有照片更好）
- 數據展示（訂閱、會員、購買人數）
- 合作夥伴 Logo（CMoney）

---

### CTA Section（行動呼籲）

**原則**
- 清楚說明下一步是什麼
- 降低風險感（免費試用、退款保證）
- 增加急迫感（限時優惠、名額限制）

**文案範例**
```
🔥 限時早鳥優惠：原價 NT$12,299 → NT$6,099（48折）

✅ 30 天無條件退款保證
✅ 分期付款 0 利率
✅ 終身觀看權限

[立即報名] （已有 482 人加入）
```

## Webinar 腳本框架

### 整體結構（60-90 分鐘）

| 時間 | 段落 | 目標 |
|------|------|------|
| 0-5 min | 開場 | 建立連結、設定期望 |
| 5-15 min | 痛點與故事 | 引發共鳴 |
| 15-40 min | 內容教學 | 提供價值 |
| 40-55 min | 產品介紹 | 展示解決方案 |
| 55-65 min | CTA | 推動行動 |
| 65-90 min | Q&A | 處理異議 |

---

### 開場（0-5 分鐘）

**目標**：讓觀眾願意留下來看完

**腳本框架**
```
「Hello 大家好，我是 Mike。

在開始之前，我想先問大家一個問題：
你願意工作一輩子嗎？

如果你的答案是『不願意』，那今天這場直播就是為你準備的。

接下來的 60 分鐘，我會分享：
1. 我如何在 4 年內達成財務自由
2. 普通人也能複製的投資方法
3. 一個特別優惠，只給今天參加的你

準備好了嗎？我們開始。」
```

---

### 痛點與故事（5-15 分鐘）

**目標**：建立信任和共鳴

**腳本框架**
```
「先講結論：我也是普通人。

[講述個人故事]
- 普通家庭出身
- 曾經是月光族
- 在中國負債累累
- 移民美國的文化衝擊

[轉折點]
- 2018 年開始投資美股
- 成為特斯拉早期投資者
- 4 年後達成財務自由

[關鍵洞察]
「老實說，我的第一感覺是...投資不難，難的是認知。
你賺不到認知以外的錢。」
```

---

### 內容教學（15-40 分鐘）

**目標**：提供真正有價值的內容

**建議主題（選擇 3-4 個）**
1. 為什麼美股適合普通人
2. 抄底策略的核心邏輯
3. 存股策略如何建立被動收入
4. ETF 配置的基本原則
5. 常見投資錯誤（Mike 親身經歷）

**教學原則**
- 每個概念用生活例子解釋
- 穿插自己的真實案例
- 適時互動（「有沒有人也遇過這種情況？」）
- 為產品介紹埋下伏筆

---

### 產品介紹（40-55 分鐘）

**目標**：自然地從教學過渡到產品

**腳本框架**
```
「我剛才講的這些方法，其實你自己也可以做。

但問題是：
- 需要花很多時間研究
- 需要自己判斷進場時機
- 沒有人可以問

所以我做了一個工具...」

[介紹產品]
- 功能逐一說明（對應前面的痛點）
- 展示實際畫面
- 強調和其他產品的差異

[社會證明]
- 已經有多少人在用
- 學員的真實回饋
```

### Q&A（65-90 分鐘）

**目標**：處理殘餘異議

**常見問題準備**
1. 我沒有很多錢，可以開始嗎？
2. 我不懂英文怎麼辦？
3. 這和你的 YouTube 內容有什麼不同？
4. 如果不滿意可以退款嗎？
5. 我應該買課程還是 APP？

## 文案語氣指南

### 要做到
- ✅ 用「我們」而不是「你們」
- ✅ 說「普通人」而不是「投資者」
- ✅ 用具體數字（4 年、43 歲、20 萬）
- ✅ 承認風險（「投資沒有人能保證獲利」）
- ✅ 講故事而不是說教

### 不要做
- ❌ 誇大報酬（「保證賺錢」）
- ❌ 貶低受眾（「你不懂投資」）
- ❌ 過度專業術語
- ❌ 壓力銷售（「今天不買就沒有」）
- ❌ 攻擊競品
