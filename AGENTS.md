# Webinar MVP Project

## 目標
根據 SPEC.md 建立 Webinar 系統的 MVP。

## 技術選型
- Frontend: Next.js + TypeScript + Tailwind
- Backend: Next.js API Routes
- Realtime: Socket.io
- Video: Video.js + HLS.js
- Database: JSON files (MVP) → PostgreSQL (production)

## MVP 範圍 (4-6 週)
1. 影片播放（無快轉）
2. 即時聊天 + 自動聊天
3. CTA 按鈕（時間觸發）
4. 觀看人數
5. 註冊表單 + 確認頁
6. 候場倒數
7. 基本後台
